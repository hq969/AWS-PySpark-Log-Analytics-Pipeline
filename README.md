# AWS PySpark Log Analytics

🔍 Real-time analytics pipeline using AWS + PySpark to analyze server logs.

## ⚙️ Tech Stack
- AWS Kinesis (Log ingestion)
- PySpark on AWS EMR (Streaming processing)
- S3 (Storage)
- Athena + Glue (Query layer)
- Streamlit (Dashboard)
